## Installation

### Dependencies
These dependencies must be present before building
 - `python 3.0+`
 - `pyside6+`
 
 You can install these on a Ubuntu-based system by executing this command:

 `sudo apt install python3 ` \
 `sudo apt install python3-pip`
 `pip3 install pyside6 `

### Executing
```
sudo python main.py
```
